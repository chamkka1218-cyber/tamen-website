아래 폴더(tamen_website_READY)를 기반으로 ㈜태멘 B2B 홈페이지를 이어서 작업하려고 한다.

목표
- 서버 없이 동작하는 정적 홈페이지(0원)
- B2B 영업/입찰/지명원 대체용
- 방문자: 건설사 구매/외주, 시공사 공무/견적, 설계/CM/감리, 발주처

현재 반영 완료 사항(요구사항)
- 히어로 문구: “지하이중벽 · 배수판 전문 솔루션”
- 회사소개 섹션: 회사 전경 사진 자리(placeholder) + 대표 연락처(전화/팩스/이메일)
- 제품/공법 용어: “불연 / 준불연”
- 제품명: “SMC · C.S 불연 TM 벽체배수판”
- 제품 소개 문구 수정(스틸 판넬 지하 이중 벽체 시스템 / 친환경 지하 이중 벽체 시스템)
- PVC 상세 문구 수정(전면 환기 구조 기반의 지하 이중 벽체 시스템 / 탈부착방식 유지관리)
- 시공사례 라벨 문구 수정(안내 사인 적용, B1,B2 등 층 표기, IN/OUT 방향 유도, 방향 표기, 포인트 벽면, 기업 로고/포인트 디자인)
- 규격 표 수정(500×500: 30T,45T,70T,120T,200T / 730×2,200: 30T,45T,70T)
- 규격 이미지 placeholder(500 3장, 730 2장)
- 자료실 섹션 유지 + PDF는 assets/에 있음

다음으로 하고 싶은 작업(우선순위)
1) 문구를 더 '입찰/공공기관' 톤으로 객관적/검토 중심으로 다듬기
2) 시공사례 사진/캡션을 실제 현장 기준으로 확장(현장명/범위 요청 가능 문구 유지)
3) 규격 섹션을 '도면·사양 요청' 흐름으로 개선

요청
- 현재 코드 구조를 유지하면서, 수정이 필요한 부분이 있으면 파일별로 diff 형태로 제안해줘.
- 가능하면 한 번에 적용 가능한 코드 블록으로 제공해줘.
